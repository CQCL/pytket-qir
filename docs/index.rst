pytket-qir
==================================

``pytket-qir`` is an extension to ``pytket`` that allows ``pytket`` circuits to
be converted to qir.

``pytket-qir`` is available for Python 3.10, 3.11, 3.12 and 3.13, on Linux,
MacOS and Windows. To install, run:

::

    pip install pytket-qir
